OpenSource | MealDB connector

This project was a winner at the Microsoft Connector Hackathon. The Meal DB it's an open online database for food meal recipes. Using recipe books is old school. There must be a better way! The connector provides high-quality transparent PNG ingredients, meal image uploads, and browsing of meals in sequence and areas, and categories. Required is a Microsoft Power Apps or Power Automate plan to access it. Supported operations include searching for a meal by name. One may search using the first single letter, look for a Meal using id, get a random meal Lookup, filter meals using the main ingredient, and list all meal categories. No Authentication is needed to access the MealsDB. You can plug in and test the connector once it is live [currently under deployment].
